
# CarSharing 

**CarSharing** — это веб-платформа, предназначенная для того, чтобы арендовывать машины на достаточно длительный срок пользования. Платформа позволяет пользователям искать подходящие машины для езды, возможно кто-то в будещем захочет купить себе такую, а также делиться мнение о машине, оставляя комментарии.

---

##  Структура приложения 

| Раздел | Маршрут | Тип доступа | Описание |
|--------|---------|-------------|----------|
| Главная | / | Публичный |Страница с машинами, тарифами и тп |
| Аутентификация | /auth/* | Публичный |  Вход, регистрация, восстановление пароля |
| Каталог машин | /catalog | Публичный | Список машин с фильтрацией по категориям, цене и т.д. |
| Страница машины | /catalog/:id | Публичный | Полное описание машины, включающее фото, описание, возможность добавить в избранное|
| Поиск | /search | Публичный |Страница с результатами поиска  |
| Личный кабинет | /account | Приватный | Личный кабинет пользователя с его данными и историей поездок и тп |
| Избранное | /favorites | Приватный | Список понравишхся пользователю машин |
| Админ-панель | /admin | Приватный |Управление списком машин, поездками, учетными записями пользователей и тд |


---


## Технологический стек 

**Хранение данных**

Данные хранятся в SQLite базе данных (`backend/data/database.sqlite`).
Используется библиотека sql.js (SQLite, скомпилированный в WebAssembly).

**Безопасность**

- Пароли хешируются через bcryptjs
- Разделение прав доступа (user/admin)


**Backend:**
- Node.js
- Express.js
- sql.js (SQLite)
- bcryptjs (хеширование паролей)
- jsonwebtoken (JWT аутентификация)
- express-validator (валидация)

**Frontend:**
- HTML, CSS
- JavaScript
- Fetch API для запросов к серверу

**Хостинг**
 - localhost:5500

## API Endpoints

### Авторизация

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/auth/register` | Регистрация нового пользователя |
| POST | `/api/auth/login` | Вход в систему |
| GET | `/api/auth/me` | Получение данных текущего пользователя |
| POST | `/api/auth/logout` | Выход из системы |

### Автомобили

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/cars` | Получить все автомобили |
| GET | `/api/cars/available` | Получить доступные автомобили |
| GET | `/api/cars/:id` | Получить автомобиль по ID |
| POST | `/api/cars` | Создать автомобиль (admin) |
| PUT | `/api/cars/:id` | Обновить автомобиль (admin) |
| DELETE | `/api/cars/:id` | Удалить автомобиль (admin) |

### Бронирования

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/bookings/my` | Мои бронирования |
| GET | `/api/bookings` | Все бронирования (admin) |
| GET | `/api/bookings/:id` | Получить бронирование по ID |
| POST | `/api/bookings` | Создать бронирование |
| POST | `/api/bookings/:id/cancel` | Отменить бронирование |
| POST | `/api/bookings/:id/start` | Начать поездку |
| POST | `/api/bookings/:id/complete` | Завершить поездку |

### Тарифы

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/tariffs` | Получить все тарифы |
| GET | `/api/tariffs/:id` | Получить тариф по ID |
| POST | `/api/tariffs` | Создать тариф (admin) |
| PUT | `/api/tariffs/:id` | Обновить тариф (admin) |
| DELETE | `/api/tariffs/:id` | Удалить тариф (admin) |

### Избранное

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/api/favorites` | Получить избранные автомобили |
| POST | `/api/favorites/toggle` | Добавить/удалить из избранного |
| DELETE | `/api/favorites/:carId` | Удалить из избранного |
